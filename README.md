# **Fritzing-parts**

Fritzing parts for the various circuits documented in this organization's repositories.

## Contents

| File | Description |
|------|-------------|
| `parts/WDC-6502.fzz` | WDC 6502 microprocessor |
| `parts/Hitachi-HM62256.fzz` | Hitachi HM62256 RAM |
| `parts/SST-SST39SF010A.fzz` | SST SST39SF010A ROM |

## Usage

1. Open [Fritzing](https://fritzing.org/).
2. Import the `.fzz` part files via **File → Open** or drag them into the Fritzing workspace.
3. Use the parts in your breadboard, schematic, or PCB views.

## Requirements

- [Fritzing](https://fritzing.org/) (version 0.9.3 or later recommended)
