# fritzing-parts

Fritzing parts for the **piclone** circuit.

This repository contains custom Fritzing part files used to design and document the piclone hardware project.

## Contents

| File | Description |
|------|-------------|
| `parts/6502.fzz` | Fritzing part for the 6502 microprocessor |
| `parts/ramchip.fzz` | Fritzing part for the RAM chip |

## Usage

1. Open [Fritzing](https://fritzing.org/).
2. Import the `.fzz` part files via **File → Open** or drag them into the Fritzing workspace.
3. Use the parts in your breadboard, schematic, or PCB views.

## Requirements

- [Fritzing](https://fritzing.org/) (version 0.9.3 or later recommended)
